# AulaConecta — entrega de hoy

## Orden obligatorio
1. Ejecuta `supabase_estabilidad_final.sql` completo en Supabase.
2. Ejecuta `verificar_supabase_final.sql` y confirma que tablas y funciones muestren `OK`.
3. Copia `.env.local` al proyecto.
4. Ejecuta `npm install` y luego `npm run check`.
5. Prueba con dos cuentas: registro, solicitud/postulación/aceptación, chat y grupo privado.
6. Publica esta misma carpeta en GitHub/Netlify.

## Correcciones críticas incluidas
- Catálogos visibles durante el registro sin sesión iniciada.
- Funciones de XP, logros y ranking incorporadas al SQL final.
- Funciones de solicitudes, tutorías, grupos, chat y descargas verificadas contra el frontend.
- Moderación muestra únicamente materiales pendientes.
- Comando `npm run check` valida compilación, lint y contrato frontend-Supabase.
